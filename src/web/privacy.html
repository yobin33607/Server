Privacy Policy

How the bot handles server and user data.

Information We Process

The bot may store limited Discord-related data needed to provide moderation, verification, and backup features, including server IDs, user IDs, moderation actions, verification state, and backup metadata.

This data is used only to operate the bot’s features and improve reliability. We do not sell or share your data with third parties for advertising purposes.

Retention and Control

Some moderation and verification records are retained in the local SQLite database while the bot is active. Server administrators control the bot’s permissions and can remove the bot from a server at any time.
